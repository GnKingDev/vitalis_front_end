# 1. Connexion et tableau de bord

## Connexion

1. Ouvrir l'application (URL fournie par l'établissement).
2. Saisir **email** et **mot de passe** du compte.
3. Cliquer sur le bouton de connexion.
4. Si la connexion réussit, l'utilisateur est redirigé vers le **tableau de bord**.

En cas d'échec, un message d'erreur s'affiche (identifiants incorrects, compte suspendu, etc.).

---

## Premier connexion / Changement de mot de passe

Si l'administrateur a imposé un changement de mot de passe à la première connexion :

1. Après connexion, une page **Changer le mot de passe** s'affiche.
2. Saisir l'**ancien mot de passe**.
3. Saisir le **nouveau mot de passe** (et confirmation si demandé).
4. Valider. La session continue et l'accès au tableau de bord est autorisé.

Un lien ou une entrée de menu **Changer le mot de passe** peut aussi être disponible pour modifier le mot de passe plus tard.

---

## Tableau de bord

- **Accès** : Menu **Tableau de bord** (visible pour tous les rôles).
- **Contenu** : Vue d’ensemble adaptée au rôle (statistiques du jour, liens rapides, etc.).
- **Actions** : Les cartes ou liens permettent d’accéder aux principales pages (liste des patients, paiements, consultation, etc.).

Le tableau de bord sert de point d’entrée ; les actions détaillées se font depuis les menus spécifiques décrits dans les chapitres suivants.
